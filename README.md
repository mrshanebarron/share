# Laravel Design Share

Social share buttons component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/share
```

## Usage

### Livewire Component

```blade
<livewire:ld-share />
```

### Blade Component

```blade
<x-ld-share />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-share-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-share-views
```
